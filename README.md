# Customer-Feedback-Analyzer
## Google Drive link to datsets: https://drive.google.com/drive/folders/1-DJH4BXvNJH2Z19_gOwc3oLhWtpeehTx?usp=sharing
## Dataset Download

When you run `customer_analyzer_preprocessing.ipynb`, the notebook  automatically downloads:
- Reviews.csv
- cleaned_reviews.csv
- pos_reviews.csv
- neg_reviews.csv

All data will be saved into the `data/` folder automatically.

## Clone Repository 
git clone https://github.com/charvimaurya/Customer-Feedback-Analyzer.git
cd Customer-Feedback-Analyzer

## Install Dependencies
pip install -r requirements.txt

## Run the notebook
-Open the notebook in Jupyter:
jupyter notebook notebooks/customer_analyzer_preprocessing.ipynb
